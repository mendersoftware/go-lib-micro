Mender: go-lib-micro/ws
==============================================

Mender supports add-ons that pass messages over websockets. This module
provides declarations and definitions that any actor in the add-on
theatre uses. 

![Mender logo](https://mender.io/user/pages/04.resources/_logos/logoS.png)


## ws/shell

[Mender-shell](https://github.com/mendersoftware/mender-shell) is a Mender add-on
allowing to run a remote terminal to any accepted device from the Mender UI.
